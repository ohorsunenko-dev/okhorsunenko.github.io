В этом практической работе реализовал функционал интернет магазина. 
Были реализованы следующие страницы и функционал:
1.Главная страница
2.Каталог товаров
2.1 Страница категории товара
2.2 Страница самого товара
3.Корзина и оформление покупок
4.Кабинет пользователя
4.1 Страница приветствия
4.2 Редактирование данных
6.Регистрация и Авторизация
7.Форма обратной связи
8.Панель администратора 

В панели Администратора были реализованы:
1. Вход в панель(Проверка прав доступа)
2. Управление товарами(добавление, показ товаров, редактирование, удаление)
3. Управление категориями(добавление, показ категорий, редактирование, удаление)
4. Управление заказами(показ хаказов, редактирование, удаление)

Использовал стандратный HTML шаблон для магазины

Также прикреплена база данных onlineshop.sql
